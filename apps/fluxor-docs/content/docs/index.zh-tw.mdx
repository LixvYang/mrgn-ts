## 🌟 快速開始

> 您可以使用 Mixin 隱私錢包和 Solana 錢包完成借貸所需的所有操作

我們目前有兩個平台：

- [Fluxor Mixin](https://app.fluxor.cc) - 使用 Mixin 隱私錢包
- [Fluxor Solana](https://app.solana.fluxor.cc) - 使用 Solana 錢包

兩個平台都提供：
- 💰 存入資產賺取原生收益
- 💳 靈活借入所需資產
- 📊 管理您的借貸倉位
- 🚀 體驗 Solana 原生智能合約
